{% blocktrans %}PiplMesh is the project of students at Faculty of Computer and Information Science of University of Ljubljana and wlan slovenija open wireless network. {% blocktrans %}You can read more about the project on development pages at wlan slovenija website.{% endblocktrans %}

{% blocktrans %}For development GitHub is used. If you have found a bug or have any other technical issue with the site, please use GitHub to report it. Source code and known issues are found there. A list of contributors is also avaible as GitHub profiles list.{% endblocktrans %}
